# CRM em Python

Sistema simples de CRM web feito em Python para cadastrar clientes e salvar dados em uma planilha do Google Sheets.

---

## Tecnologias
- Python 3  
- Flask  
- Google Sheets API  
- HTML + CSS  

---

## Funcionalidades
- Login de usuário  
- Cadastro de clientes  
- ID automático  
- Data de cadastro  
- Responsável pelo cliente  
- Banco em nuvem (Google Sheets)
